# To-Do App

Simple to-do application that enables adding, crossing out, and deleting items. Created with React, TailwindCSS, Node.js, and DaisyUI.

## Built With
- https://daisyui.com/
- https://tailwindcss.com/
- https://react-icons.github.io/react-icons
- https://react.dev/
- https://nodejs.org/en
